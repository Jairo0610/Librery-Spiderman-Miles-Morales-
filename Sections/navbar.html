Navbar

Barra de navegación que contiene imagen, enlaces, dropdown y barra de búsqueda.

Barra de Navegación Completo.

  • Link
  • Link
  • Dropdown
    • Subelemento 1
    • Subelemento 2

Genera esta barra de navegación con el siguiente código.

CÓDIGO

        <nav class="menu">
            <div class="menu__imagen"> <!--Contenedor para imagen (logo)-->
                <img src="#" alt="logo" >
            </div>
            <ul class="menu__nav"> <!--Contenedor para enlaces principales-->
                <li class="nav__elemento"><a href="#">Link</a></li>
                <!--Coloca los enlaces como la línea anterior aquí-->
                <li class="nav__elemento--dropdown"> <!--Este es un elemento dropdown-->
                    <div class="dropdown__boton">
                        <span>Dropdown</span>
                        <div class="flecha"><div></div></div>            
                    </div>

                    <ul class="dropdown__lista">
                        <li class="nav__elemento"><a href="#">Subelemento 1</a></li>
                        <!--Coloca los subelementos de enlaces como la línea anterior aquí-->
                    </ul>   
                </li>
            </ul>

            <form role="search" class="menu__busqueda">
                <input type="text" class="busqueda--barra" name="" id="">
                <input type="button" class="busqueda--boton" name="" id="" value="Buscar">
            </form>
        </nav>
